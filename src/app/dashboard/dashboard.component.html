Welcome to Angular Forms! Angular provides two kind of forms:
  • Template-driven Forms
  • Reactive Forms

Template-driven Forms

Recommended for rather simple forms without any complex validation.
Link to Angular Documentation

Reactive Forms

Useful for more complex forms, like:
  • Validations going beyond simple validations like if a field is required or if there are at least / no more than X characters:
    • Validation dependent on multiple fields
    • Regex Validation
  • Multiple validations errors shown on the UI (better code maintainability)
  • If fields may have to be shown or hidden dynamically on the user input
  • If questions are served through a service -> See Dynamic Forms
