How it works

This repository consists of several branches. One for each exercise as well as one for each solution. You are currently on the SOLUTION-5 branch. In order to progress, please open the corresponding exercise branch and implement your code there. Afterwards, you can verify your solution against the corresponding solution branch.

Example form

The Example form contains all elements which you learn during this workshop in one form:

  • FormGroup
  • FormControl
  • FormArray

Moreover, it uses advanced Angular features like components with an attribute as a selector as well as @ContentChild and @HostBinding, which allow us to dynamically add and remove input errors more easily on the form. Those features are for simplicity reason not part of this workshop, as if you know those, likely also now how to write complex forms appropriately.
